

1 - Make a copy of the folders css and pics and the file index.php from your GLPI;


2 - Copy the css, fonts and pics folders and the index.php file of your files uncompressed for GLPI;


3 - To modify the template of the email notification sent by GLPI follow the instructions in the file notification.txt;


4 - To change the logo create your own image with size 200px x 200px, named to logo_big.png
 and copy to glpi/pics folder.


 OR:
 change the GLPI_THEME_PATH parameter in config/define.php to use the desired theme
